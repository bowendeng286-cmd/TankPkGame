# 触摸控制器自定义功能使用指南

## 功能概述

触摸控制器自定义功能允许触摸设备用户自定义游戏中摇杆和开火键的位置、大小和参数，以获得最佳的游戏体验。

## 如何访问

1. 在主菜单点击 **[设置]**
2. 在设置页面点击 **控制器设置**（仅触摸设备显示）
3. 进入控制器配置界面

## 功能说明

### 模式选择

- **单人模式**：配置单人游戏时的控制器布局（左下摇杆 + 右下开火键）
- **双人模式**：配置双人游戏时的控制器布局（P1: 左上摇杆+左下开火，P2: 右上摇杆+右下开火）

### 调整位置

1. **长按**控制器（摇杆或开火键）
2. 拖动到目标位置
3. 松手完成调整

### 调整参数

1. **点击**控制器（摇杆或开火键）
2. 弹出参数调节面板
3. 拖动滑动条调节参数：
   - **摇杆大小**：40-80px（外圈半径）
   - **死区范围**：0-20px（静止判定范围，越小越灵敏）
   - **按钮大小**：30-60px（开火键半径）
4. 点击 **[确定]** 保存参数，或 **[取消]** 放弃修改

### 保存配置

- 点击 **[保存]** 按钮保存当前配置
- 配置会自动保存到浏览器 localStorage
- 下次打开游戏时自动加载

### 重置默认

- 点击 **[重置默认]** 按钮恢复出厂设置
- 所有位置和参数恢复到默认值

## 配置参数说明

### 摇杆参数

- **外圈半径**：摇杆底座的大小，越大越容易操作
- **内圈半径**：自动计算（外圈半径 × 0.42）
- **最大距离**：摇杆可移动的最大距离（外圈半径 × 0.83）
- **死区范围**：中心静止区域，手指在此范围内不会触发移动
  - 死区越小，灵敏度越高
  - 死区越大，越容易保持静止

### 开火键参数

- **按钮半径**：开火键的大小，越大越容易点击

## 使用技巧

1. **位置调整**
   - 根据手指自然位置调整控制器
   - 避免放置在屏幕边缘，防止误触
   - 双人模式注意两个玩家的控制器不要重叠

2. **大小调整**
   - 手指较大的用户可以增大控制器尺寸
   - 屏幕较小的设备建议减小控制器尺寸

3. **死区调整**
   - 新手建议增大死区（5-10px），更容易控制
   - 高手建议减小死区（0-3px），提高灵敏度
   - 如果坦克经常"抖动"，增大死区

## 配置存储

- 配置保存在浏览器 localStorage
- 键名：`tankgame_controls`
- 清除浏览器数据会丢失配置
- 不同浏览器的配置独立存储

## 故障排除

### 控制器不显示

- 确认设备支持触摸（`ontouchstart` 事件）
- 检查浏览器控制台是否有错误
- 尝试刷新页面

### 配置无法保存

- 检查浏览器是否禁用了 localStorage
- 尝试清除浏览器缓存后重试
- 检查是否处于隐私/无痕模式

### 控制器位置异常

- 点击 **[重置默认]** 恢复出厂设置
- 检查屏幕方向（横屏/竖屏）
- 尝试重新调整位置

## 技术细节

### 配置数据结构

```javascript
{
  version: 1,
  singlePlayer: {
    joystick: { x, y, outerRadius, innerRadius, maxDistance, deadZone },
    fireButton: { x, y, radius }
  },
  dualPlayer: {
    player1: { joystick: {...}, fireButton: {...} },
    player2: { joystick: {...}, fireButton: {...} }
  }
}
```

### 参数范围

- 摇杆外圈半径：40-80px
- 摇杆死区：0-20px
- 开火键半径：30-60px

### 自动计算

- 摇杆内圈半径 = 外圈半径 × 0.42
- 摇杆最大距离 = 外圈半径 × 0.83
